# Exploratory Data Analysis (EDA) on Dataset for Data Analytics
## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a cleaned dataset to understand its structure, identify trends, detect outliers, and extract meaningful insights. 
EDA helps analysts understand data behavior before applying advanced analytics or machine learning techniques.
## Objectives
* Understand the structure and characteristics of the dataset
* Calculate descriptive statistical measures
* Identify trends and patterns in the data
* Detect potential outliers
* Analyze sales and order trends
* Generate meaningful insights
## Tools Used
* Python
* Pandas
* Jupyter Notebook
* Microsoft Excel
## Dataset Information
* Dataset Name: Cleaned Dataset for Data Analytics
* File Format: Excel (.xlsx)
* Domain: Data Analytics
## EDA Tasks Performed
### Descriptive Statistics
* Mean
* Median
* Count
* Minimum and Maximum values
### Trend Analysis
* Product Sales Analysis
* Daily Sales Analysis
* Monthly Sales Analysis
* Monthly Order Analysis
* Order Status Analysis
* Payment Method Analysis
### Outlier Detection
* Identified unusual values using statistical methods
* Analyzed their impact on the dataset
## Results
* Statistical characteristics of the dataset were analyzed.
* Product sales trends were identified.
* Daily and monthly sales patterns were examined.
* Order status distribution was analyzed.
* Payment method preferences were identified.
* Outliers were detected and evaluated.
## Key Insights
* Product sales varied across different products.
* Sales and order volumes changed over time.
* Certain payment methods were preferred by customers.
* Statistical analysis provided valuable business insights.
* Potential outliers were identified in numerical columns.
## Conclusion
Exploratory Data Analysis was successfully performed on the dataset. The analysis revealed important patterns, trends, and outliers that can support business decision-making and future data analytics projects.
