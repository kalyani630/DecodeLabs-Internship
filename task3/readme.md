# SQL Data Analysis Using SQLite3 on Dataset for Data Analytics
## Project Overview
This project focuses on performing SQL Data Analysis on a cleaned dataset using SQLite3. The dataset was imported into a SQLite database, and various SQL queries were executed to retrieve, filter, group, sort, and summarize data. The project demonstrates how SQL can be used to transform raw data into meaningful insights for decision-making.
## Objectives
* Create a SQLite database for storing the dataset
* Import the cleaned dataset into a database table
* Retrieve data using SQL queries
* Filter records using SQL conditions
* Group records using SQL aggregation techniques
* Calculate summary statistics using aggregate functions
* Derive meaningful insights from the dataset

### Data Retrieval
* Retrieved all records using SELECT statements
* Retrieved specific columns for analysis
### Data Filtering
* Applied WHERE conditions to extract relevant records
* Filtered data based on specific criteria
### Data Grouping
* Used GROUP BY to categorize records
* Generated category-wise summaries
### Aggregate Analysis
* COUNT() for total records
* SUM() for total values
* AVG() for average values
### Data Sorting
* Used ORDER BY to sort records
* Displayed data in ascending and descending order
